Detailed Description

Each piece of road has two objects of the type SequenceInfo. One which contains the predecessors of the road piece for all lanes and one which contains the successors of the road piece of all lanes. The object SequenceInfo contains all needed information of the predecessor/successor, which is needed to connect the road piece to them.
